# 视频脚本题词器

一个独立的静态题词器页面，入口文件是 `index.html`。

## 功能

- 粘贴、导入、导出视频脚本
- 自动保存脚本和显示设置
- 播放、暂停、回到开头、前进后退
- 双击台词段落，从该段附近开始滚动
- 调整滚动速度、字号、行距
- 黑底、白底、棚拍三种主题
- 镜像显示、焦点线、全屏、防息屏

## 快捷键

- 空格：开始或暂停
- 上下方向键：调整速度
- 左右方向键：微调位置
- F：全屏

直接打开 `teleprompter/index.html` 即可使用。

## 手机使用

没有局域网时，把仓库发布到 GitHub Pages 或其他静态托管服务，然后用手机访问：

```text
https://你的用户名.github.io/仓库名/teleprompter/
```

在手机浏览器里打开后，可以添加到主屏幕。脚本和设置会保存在当前浏览器里。
